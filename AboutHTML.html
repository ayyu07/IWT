<!DOCTYPE html>

-This tag tells about the version of the html

<html>

-This tag tells that the given text is of html type

<head>

- It is the container for metadata (data about data) and is placed between the <html> tag and the <body> tag.

<title>

-This tag defines the title of the document.

<body>

-This tag contains all the contents of an HTML document, such as text, hyperlinks, images, tables, lists, etc.

<p>

-This tag defines a paragraph.

<h1-h6>

-h1 defines the most important heading while h6 defines the least important heading

<img src="" alt="">

-images are defined with the <img> tag.The src attribute specifies the URL (web address) of the image

<a href>

-This tag defines a hyperlink, which is used to link from one page to another.

<style>

-This tag is used to define style information for an HTML document.

<table>

-This tag defines an HTML table.

<tr>

-This tag defines a row in an HTML table.

<td>

-This tag defines a standard cell in an HTML table.

<th>

-This tag defines a header cell in an HTML table.

<hr>

-This tag defines a break in an HTML page (e.g. a shift of topic).

<br>

-This tag inserts a single line break.

<strong>

-This tag makes the text font bold

<del>

-This tag cuts the particular font

<mark>

-This tag makes the text heighlighted
